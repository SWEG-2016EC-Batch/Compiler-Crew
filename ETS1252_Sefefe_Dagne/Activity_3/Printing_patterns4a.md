Loop i from 1 to 5

### 3.Pattern 2: 4x6 Grid of Letters

Loop i from 0 to 3

Print letter

### 4.Pattern 3: Right-Angled Triangle of Numbers

Loop i from 1 to 5

Loop j from 1 to i

Print j

New line

Loop i from 1 to n1

Print j

### 6.Pattern 5: Capital Letters Pattern

Loop i from 1 to n1

Initialize letter

Print letter

New line

### 7.Pattern 6: Lowercase Letters Pattern

Loop i from 1 to n1

Print letter

### 10.Pattern 9: Hollow Inverted Half Pyramid

Loop i from n3 to 1 (decrement)

Print "*" at borders
